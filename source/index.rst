Welcome to the Extractor_flow user documentation!
=============================================

.. note::
    New release available: 1.0.0.

Extractor_flow pipeline is developed by the Sherbrooke Connectivity Imaging Lab (`SCIL`_).
The pipeline is based on Nextflow and Singularity. 
.. _SCIL: http://scil.usherbrooke.ca/en/

Use Extractor_flow in published works should be accompanied by the following citation:

T.B.A

Other citations can be added if Extractor_flow is used in a publication. Please see :ref:`references`

For Linux users, please see this section :ref:`singularity-extractor_flow` for setup.

For MacOS users, please see this section :ref:`docker-extractor_flow` for setup.

For any issues or difficulties with Extractor_flow, please use our Neurostar tag: https://neurostars.org/tag/Extractor_flow

.. tip::
    If you want to analyse datasets with white-matter lesions use profile ABS.
